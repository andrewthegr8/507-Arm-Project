Compute internal motor conversion parameters based on microstepping and gearing.

Call at runtime once per motor to compute the conversions between radians and controller step units, and to populate acceleration/velocity conversion factors stored in motor.

Parameters
motorPointer to the motor configuration to populate.
microstepMicrostepping multiplier (e.g. 1, 2, 4, 8, ...).
gearboxRatioGearbox ratio (motor revolutions per output revolution).
stepsPerRevSteps per motor revolution (full steps, e.g. 200).

Execute the provided trajectory on the given motors.

Execute a trajectory from its first waypoint through completion.

Parameters
motorConfigsArray of motor configuration structures.
trajectoryPointer to the trajectory to execute.

Execute the provided trajectory on the given motors.

This function blocks until the entire trajectory has been executed. It handles any initial servo command, sends the initial joint commands, then repeatedly waits until the current waypoint is reached before advancing to the next waypoint.

Parameters
motorConfigsPointer to an array of motor_config_t (length >= NUM_MOTORS).
trajectoryPointer to the trajectory_t to execute.

Zero the motor positions for a set of motors.

Useful for homing procedures where the current physical position should be declared as the zero reference. Temporarily switches the controller to velocity mode to allow direct modification of the XACTUAL register.
